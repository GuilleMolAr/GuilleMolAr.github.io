Normalización de Localidades con Python y Excel

Descripción del Proyecto

Automatización del proceso de normalización de datos de localidades utilizando Python y Excel para estandarizar las entradas de direcciones y localidades.

Detalles Técnicos

  • Uso de Pandas para procesar grandes cantidades de datos en Excel
  • Normalización de nombres de localidades y eliminación de duplicados
  • Automatización del proceso de validación

Resultados

Incremento en la precisión de los datos y reducción de entradas duplicadas en el sistema de base de datos.
