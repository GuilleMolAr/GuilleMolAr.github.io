Normalización de Localidades con Python y Excel

Este proyecto consistió en la normalización de nombres de localidades en un archivo Excel basados en sus códigos postales. Se utilizaron técnicas de análisis y transformación de datos para corregir y estandarizar las entradas erróneas o inconsistentes.

Objetivos

  • Corregir nombres de localidades basados en sus códigos postales.
  • Automatizar el proceso de normalización con Python y Excel.

Herramientas utilizadas

  • Python
  • Pandas
  • Excel VBA

Resultado

La normalización permitió la corrección automática de los datos, eliminando errores manuales y mejorando la calidad del dataset.
