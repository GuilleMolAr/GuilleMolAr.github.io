Optimización de Rutas con Python

Descripción del Proyecto

Algoritmo para optimizar la asignación de rutas en entregas con múltiples zonas y unidades, utilizando coordenadas de latitud y longitud para minimizar el tiempo y distancia de los recorridos.

Detalles Técnicos

  • Uso de Python y OR-Tools para la optimización de rutas
  • Consideración de restricciones de volumen y capacidad por vehículo
  • Visualización de rutas optimizadas en un mapa

Resultados

Reducción en el tiempo total de entrega y optimización en el uso de recursos logísticos.
