Optimización de Rutas con Python

Este proyecto consiste en la optimización de rutas de distribución basadas en zonas, volumen de carga y capacidad de las unidades de distribución. Se usaron algoritmos de optimización para generar rutas eficientes que minimicen la distancia y el tiempo.

Objetivos

  • Generar rutas de distribución óptimas basadas en coordenadas geográficas.
  • Minimizar la distancia recorrida por las unidades de distribución.

Herramientas utilizadas

  • Python
  • Pandas
  • Google Maps API
  • Ortools

Resultado

Se lograron rutas más eficientes, lo que permitió una reducción de costos y tiempos de entrega.
