Control de Envíos con Google Sheets

Este proyecto se centró en la creación de un sistema para gestionar el control de envíos utilizando Google Sheets. El sistema permite que los envíos se registren mediante un lector de códigos de barras y marca automáticamente los envíos recibidos.

Objetivos

  • Implementar un sistema de control automatizado de envíos.
  • Utilizar Google Sheets como base para la gestión de datos.
  • Integrar el uso de lectores de código de barras.

Herramientas utilizadas

  • Google Sheets
  • Macros de Google Apps Script
  • Lectores de códigos de barras

Resultado

El sistema permitió un control eficiente y automatizado del inventario y mejoró significativamente la precisión del registro de envíos.
